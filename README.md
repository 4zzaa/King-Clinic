# King Clinic - Dental System

## 📌 Deskripsi
King Clinic adalah aplikasi berbasis web yang memiliki fungsi untuk pasien melakukan booking melalui web tanpa harus menunggu ke kliniknya.

## 🛠️ Tech Stack 
- **Language:** PHP 
- **Database:** MySQL
- **Design:** ERD, DFD, Use Case Diagram

## 🚀 Fitur Utama
1. Dashboard Admin & Dokter
2. Login register pasien 
3. Penjadwalan Konsultasi
4. Riwayat Rekam Medis Gigi
